- **기술**: Jenkins, Github Action, AWS CodePipeline, Gitlab Runner,
- **산출물**: 배포 시간을 50% 단축시키고, 배포 과정에서의 오류를 30% 감소시킨 뛰어난 결과를 달성했습니다.

- **기술**: Jenkins, Docker, Kubernetes, AWS
